你应该判断下有没有数据吧。。要不第一次访问就是没有数据的,而且all是sql的关键字,用[]括起,获取提交的数据键名称是字符串。
<code>   require_once 'conn.php';
    $id = $_POST["userid"];
    $name = $_POST["username"];
    $content = $_POST["content"];
    if($id){
      $sql = "insert into [all](userid,name,content)values('$id','$name','$content')";
      $res = mysql_query($sql);
      if(!$res)
      {
         echo "数据添加失败!"; 
      }
    }
</code>
