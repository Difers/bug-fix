创建 onItemClickListener 时的问题
我想在ListView中添加一个onItemClickListener 
<code>streamListView.setAdapter(this.streamAdapter);
            this.streamListView.setOnItemClickListener(new OnItemClickListener() {
                @Override
                public void onItemClick(AdapterView<?> a, View v, int position, long id) {
                    Intent intent = new Intent(this, MessageDetailActivity.class);
                    startActivity(intent);
                }
            });
</code>
但是得到错误提醒：
<code>The constructor Intent(new AdapterView.OnItemClickListener(){}, Class<MessageDetailActivity>) is undefined
</code>
如何解决这个问题呢？
