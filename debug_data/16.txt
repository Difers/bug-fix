android菜单出现了问题
在android2.2+版本中，想要建立一个安卓系统，用到硬件菜单按钮，但是运行后按钮一点反应也没有。
/res/menu/optionsmenu.xml
<?xml version="1.0" encoding="utf-8"?>
<code><menu xmlns:android="http://schemas.android.com/apk/res/android">
    <item android:id="@+id/menuHelp"
          android:title="Help"
          />
    <item android:id="@+id/menuShowInstallationCode"
          android:title="Show Installation Code"
          />
</menu>
</code>
activity类中
<code>@Override
public boolean onCreateOptionsMenu(Menu menu){
    MenuInflater inflater = getMenuInflater();
    inflater.inflate(R.menu.optionsmenu, menu);
    return(super.onCreateOptionsMenu(menu));
}


public boolean onOptionsItemSelected(MenuItem item){
    switch (item.getItemId()){
        case R.id.menuHelp :
            return true;

        case R.id.menuShowInstallationCode :
            return true;

        default:
            return super.onContextItemSelected(item);
    }
}
</code>
现在的结果就是按了菜单键，没有反应，也没有报错。
请大牛帮忙指导~ 
----------------------------------------------------------------------------------------------------
你要在onCreateOptionsMenu()返回真值,然后在onOptionsItemSelected(MenuItem item)中返回super.onOptionsItemSelected(MenuItem item)。代码如下:
<code>@Override
public boolean onCreateOptionsMenu(Menu menu){
    MenuInflater inflater = getMenuInflater();
    inflater.inflate(R.menu.optionsmenu, menu);
    return true;
}


public boolean onOptionsItemSelected(MenuItem item){
    switch (item.getItemId()){
        case R.id.menuHelp :
            return true;

        case R.id.menuShowInstallationCode :
            return true;

    }
    return super.onOptionsItemSelected(item);
}
</code>
