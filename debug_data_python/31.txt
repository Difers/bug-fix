def o():
    n = 1
    while True:
        n = n + 2
        yield n
next(o())

为什么用next(o()) 生成的不是3579  而是一直是3

----------------------------------------------------------------------------------------------------
next(o())  每次都创建了一个新的生成器. 改成只创建一次.
