问题描述：
append python 基础问题
a=[]
for i in range(7):
    a.append(i)
        print(a)
我知道上面是正确的
为什么
a=[]
for i in range(7):
    a=a.append(i)
        print(a)
就会报错 
----------------------------------------------------------------------------------------------------
修改方案：
append没有返回值,所以不能传给a啊。
你非要那样,需要写一个带有返回值的函数:
<code> def myappend(a,b):
  a.append(b)
  return a #注意这一行,去掉的话,只能myappend(a,i)

a=[]
for i in range(7):
  a = myappend(a,i)
print(a)
</code>
