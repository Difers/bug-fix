<code>import XXX

      needNum=666

那么我到底该如何获取参数needNum呢？之前一直试着把needNum当成类变量调用，显示失败，现在有点懵，求帮助T T

修改方案：
class A:
    def init(self, num):
        self.needNum = num
    def uselessFunctionA(self):
        print(self.needNum)
a=A(2)
a.uselessFunctionA()
